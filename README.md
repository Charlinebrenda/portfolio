# Portfolio — Yamenjeu Tchaweu Charline Brenda

Portfolio personnel développé avec Next.js et Tailwind CSS.

## Installation

```bash
npm install
npm run dev
```

## Structure

- `src/dictionaries/` — textes FR et EN
- `src/data/` — données (projets, compétences, expériences...)
- `src/sections/` — sections de la page
- `src/components/` — composants réutilisables
- `public/` — images, CVs, certifications

## Fichiers à personnaliser dans `public/`

- `CV_CHARLINE_BRENDA_FR.pdf` et `CV_CHARLINE_BRENDA_EN.pdf`
- `images/profile-photo.jpg` et `images/about-photo.jpg`
- `images/projects/` — captures de tes projets
- `images/certifs/` — captures de tes certifications
- `certifs/` — PDFs de tes certifications
